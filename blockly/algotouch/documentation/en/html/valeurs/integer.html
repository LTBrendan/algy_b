Integer

Allow the user to create integer-typed values. Here are some important points :


  • This is not a variable that can be changed. It's only a value that can be used to compare with a variable created by the user.

  • It as a limited size 2147483647 and its minimum is -2147483648. Be careful about processing time and size error.

  • Only integer can be typed in. If an unsupported character is typed in, the last correct value will be used.

This block only accept one entry : a user-typed character.


For developers

The output for this block is

  • integer

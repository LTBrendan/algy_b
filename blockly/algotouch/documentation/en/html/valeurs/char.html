Character

Allow the user to create char-typed values. Here are some important points :


  • This is not a variable that can be changed. It's only a value that can be used to compare with a variable created by the user.

  • Its length is only one. Indeed, it is a character, not a String. Even if you type in more than one character, only the first one will be kept.

  • You are free to use either alphabetical, numerical or special characters (as &, %, " ...)

This block only accept one entry : a user-typed character.


For developers

The output for this block is

  • character

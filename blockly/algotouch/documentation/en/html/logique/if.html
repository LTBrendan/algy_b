If...Then...Else

If…Then…Else is a conditionnal block :

  1. If ‘fullfilled condition’
  2. Then ‘perform action’
  3. Else ‘perform another action’

The argument situated after IfAllow to compare easily to values with types integer ou caracter. It can be simple value, or variables created by user.


Place the two values to compare in the different entry point.


The first state argument is situated after Then. It will only be exectuted if the condition is true


The second state argument is situated after Else. It won't be executed unless the condition is false.

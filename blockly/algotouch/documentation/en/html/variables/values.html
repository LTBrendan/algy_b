Values

In order to be able to program interesting algorithms, it is possible to create "local" variables. These can be used throughout the program, modified and used with arrays.


The type of the variable must be specified when it is created. It can be a character, an integer, an array's index (so it will have integer type) or can be defined as a constant, that is an integer that can not be modified.



As part of an integer or character value, it is possible to modify the value during the execution of the program, or request the value from the user. The values created can of course be deleted or renamed at will.



A "popup" is launched to inform the name of the variable, and possibly its value (as part of a constant). If the variable is an index, the array's name will be demanded if several exist.


For more information about array access, read the Getter | Setter page.
For developers

The created values are of type :

  • var_char for a character
  • var_integer for an integer

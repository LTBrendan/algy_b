Arrays

The Algo Touch software allows the creation of arrays, in order to be able to carry out simple operations on them. It is possible to perform value searches, deletions of duplicates ...


However, the type of the array must be specified when it is created. Indeed, it can contain characters (char) or integers (integer).



Once the array is created, several actions are possible. Indeed, it is possible to delete it as well as to rename it. You can also modify it. It means you can specify a new length or shuffle new values into it. It prevents the creation of macro to modify existing array. Other actions you can do with an array are referenced in the Length and Getter | Setter pages.


For developers

The created arrays are of type :

  • char_array for an array of characters
  • integer_array for an array of integer

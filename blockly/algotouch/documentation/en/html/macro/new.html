New macro

This block helps to create a new macro (loop), which could be use (call) by another macro.


Name : name given to the macro by the user, to call it after.


From : All actions to perform before starting the main loop of the macro.


Until : Instructions of the loop continue to be executed, until it's not completed. It's type : boolean.


Loop : Instructions to perform until the exit condition is not completed


Terminate : Instructions to perform after the main loop of the macro.

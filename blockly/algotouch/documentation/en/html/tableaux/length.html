Length

This block allows the user to know the size of an array he has previously created. It can be useful in particular to make a loop browsing all or part of a table.


The input argument corresponds to the block of the array whose size is wanted. So he accepts the types char_array and integer_array, which are the two types possible tables.

The block therefore returns a value of type integer.

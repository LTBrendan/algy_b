Getter | Setter

Getter

This block is used to get a value through its index.


In the first field you place the array where to get a value.


In the second field you place the index. It can be both an integer, or an index previously created and associated with this array.


Setter

This block is used to set a value through its index.


In the first field you place the array where to set a value.


In the second field you place the index. It can be both an integer, or an index previously created and associated with this array.


In the last field you place the new value to set in. It can have the types : char, var_char, integer, var_integer, operation_number or index.

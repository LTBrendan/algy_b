User manuel

Get started with AlgoTouch

You will find below a manual allowing you to apprehend at best the different functionality of this software.

Starting software

When you start AlgoTouch, a window similar to the one below should appear (after the loading screen).

Let's divide this window in different parts :

Blocks menu

You will find in the block menu all the available blocks to create your algorithms. They are sort by categories :

  • Logic
  • Maths
  • Values
  • Arrays
  • Variables
  • Macros

You will find in this documentation details block by block (accessible via the menu located on the left side of this page).

Workspace (grid)

The workspace (or espace de travail in french), is the place where you will create your programs.


It works with drag & drop. Indeed, all you need is to find out the block you need, from the block menu, to drag it to its place to be in the workspace and to drop it. To remove a block, just drag it into the bin situated in the bottom-right-corner of the workspace.


There are also two other buttons next to the bin. It allows you to zoom in or out to see only what you need. The last icon allows you to center your point of view on your programme.

Toolbar

The toolbar gives you access to different functionality :

  • Documentation : The first one open the documentation
  • Export to text : You can export your code in text format, in pseudo code.
  • Save as : You can save your programs in order to use it later.
  • Open : You can open programs you saved before.
  • Play : Execute the programs present in the workspace.
  • Language : You can change the language (french / english). The language is automatically initialized when you launch AlgoTouch with your default system language.
Tracking variables

This part of the screen allows you to visualize the variables you've created. You can find 2 differents tabs :

  • Arrays : it shows you all the arrays created. There is 3 columns :
    • Name : The name of the variable (update if renamed).
    • Index : The index of the value on the array.
    • Current Val : The current value of every array case after the run.

    By default the table is reduced to [...] but by clicking on it, you can view the content and vice versa.

  • Variables : it shows all the created variables. There is 3 columns too :
    • Name : The name of the variable (update if renamed).
    • Initialization : The value of the variable at the beginning of the program.
    • Current Val : The current value of the variable after execution
Macro block "main"

In this table you can look after the different variables you created, their name and their value. There is 2 tables with distincts columns:

  • Arrays : This tables display all the arrays you created with the informations below :

    • Name : The array's name (updated if renamed)
    • Index : The index of the value in the array
    • Current val : The current value of the box after running a macro

    By default, the fdisplay is reduced to "[...]" but you can develop or hide the content bu clicking on it.

  • Variables : This table display the different variables you instantiated. It's also compose of different columns :

    • Name : The variable's name (updated if renamed)
    • Initialization : The value of the variable before running the programe
    • Current val : The value of the variable after running the programe
Create a program

This software is based on "drag & drop" principle. So you just need to drag your blocks from the menu to the workspace.

Running

To run a program on AlgoTouch, just click the play button in the toolbar. The main macro will be runned by default. To run only a part of your program, select the macro you want before clicking the play button.

Save

Once your program crated, you can save to end it later or send to someone else using AlgoTouch.Your program will be saved in a file with ".algy" extension based on xml. You won't be able to open it in another software but AlgoTouch.

Open a program

Just like the save, it's possible to open a program previously saved. The file must have the ".algy" extension. If the file has been modified, the reading will probably fail.


After loading the file, all the blocks will be restaured (in the same place where you saved it. The variable table will also be updated.

Change language

At the moment, only 2 languages are available on AlgoTouch:

  • French
  • English

When you start the application, the default language of your device will be recovered then AlgoTouch will adapt the language to use. If you're using an unsupported language, french will be loaded by default.

What if an error occur when running a program ?

If there is any error, the message is recovered and displayed be keeping only the important part of the Java's StackTrace.

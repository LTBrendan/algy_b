Nouvelle macro

Ce bloc permet de créer une nouvelle macro (boucle), qui peut être utilisée (appelée) a partir d’une autre macro.


Nom : nom de la macro donné par l’utilisateur, pour l’appeler par la suite.


Initialisation : toutes les actions à effectuer avant la boucle principale de la macro.


Répéter jusqu'à : Condition de continuité de la boucle. Tant qu’elle n’est pas complétée, le contenu de la boucle continue à être exécuté. Elle est de type boolean.


Faire : ensemble d’actions réalisés tant que la condition de continuité n’est pas validée.


Pour finir : Ensemble d’actions réalisés après la boucle principale de la macro.

Pour les développeurs

Pour ce bloc, le type de sortie est :

  • macro

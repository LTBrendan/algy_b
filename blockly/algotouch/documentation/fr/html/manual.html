Manuel d'utilisation

Bien démarrer avec AlgoTouch

Vous trouverez dans les sections ci-dessous un manuel vous permettant d'appréhender au mieux les fonctionnalités du logiciel.

Démarrage du logiciel

Lorsque vous lancez AlgoTouch, une fenêtre similaire à celle-ci devrait apparaitre (au terme du chargement).

Divisons cette fenêtre en différentes parties :

Menu des blocs

Vous trouverez dans le menu des blocs l'ensemble des blocs disponibles pour la création de vos programmes. Ils sont triés par catégories :

  • Logique
  • Maths
  • Valeurs
  • Tableaux
  • Variables
  • Macros

Vous trouverez dans cette documentation des détails blocs par blocs (accessible via le menu situé sur le côté gauche ce cette page.

Espace de travail (grille)

L'espace de travail (ou workspace en anglais), est l'endroit où vous allez créer vos programmes.


Il fonctionne par "drag & drop" (glisser - déposer). En effet, il suffit de trouver le bloc qu'il vous faut à partir du menu des blocs, de le glisser jusqu'à l'endroit voulu sur le workspace et de le lâcher. Pour supprimer un bloc, il faut le glisser jusque l'icône "poubelle" située en bas à droite de l'espace de travail.


Vous pouvez voir aussi que l'on trouve des icones sur cet espace de travail (en bas à droite). Ils servent à zoomer et dézoomer sur vos blocs, vous permettant de gérer ce que vous voulez voir. L'icône corbeille permet de supprimer un bloc. La dernière icone permet de se recentrer sur le programme.

Barre d'outils

La barre d'outils vous donne accès à de nombreuses actions :

  • Documentation : La première icone vous permet d'accéder sur la documentation
  • Exporter au format texte : Cette icône permet d'exporter le programme dans un format texte, en pseudo code.
  • Sauvegarder : Vous pouvez sauvegarder vos programmes afin de les modifier ultérieurement.
  • Ouvrir : Vous permet d'ouvrir un programme que vous avez sauvegardé.
  • Play : Lancer l'exécution du programme que vous avez écrit.
  • Langue : permet de changer la langue de l'interface (français / anglais). Il faut savoir que la langue du logiciel au lancement est paramétrée en fonction de la langue du système d'exploitation (par défaut : français).
Suivi des variables

Cet espace vous permet de visualiser les variables que vous avez créé, leur nom ainsi que leur valeur. Vous trouverez dans cet espace deux tableaux distincts avec différentes colonnes :

  • Tableaux : Ce tableau affiche les différents tableaux que vous avez créé. IL comporte plusieurs colonnes :

    • Nom : Le nom de la variable (se met à jour si la variable est renommée)
    • Indices : L'indice de la valeur dans le tableau
    • Val courante : La valeur courante des cases du tableau après exécution

    Par défaut le tableau est réduit à [...] mais en cliquant dessus, vous pouvez en visualiser le contenu et inversement.

  • Variables : Ce tableau affiche les différentes variables que vous avez créées. IL comporte plusieurs colonnes :

    • Nom : Le nom de la variable (se met à jour si la variable est renommée)
    • Initialisation : La valeur de la variable au début du programme
    • Val courante : La valeur courante de la variable après exécution
Bloc macro "main"

C'est le bloc principal de tout programme. Il ne peut pas être supprimé, et il sera appelé par défaut lors de l'exécution.


Cependant son exécution peut être "évitée", en sélectionnant sur une autre macro avant de lancer l'exécution.


Vous retrouverez toute la documentation concernant ce bloc sur pas page correspondant aux macros.

Création d'un programme

L'application utilise le principe de "drag & drop" pour fonctionner. Il suffit donc de glisser les blocs depuis le menu des blocs jusqu'au workspace.

Exécution

Pour exécuter un programme écrit sur AlgoTouch, il suffit de cliquer sur le bouton "play" (dans la barre d'outils). C'est alors la macro main qui sera lancée par défaut. Pour lancer seulement une partie du programme, il suffit de sélectionner la macro avant de lancer l'exécution.


Sauvegarde

Une fois que vous avez créé votre programme, vous pouvez le sauvegarder, afin de le terminer plus tard, ou le transmettre à une autre personne qui utilise AlgoTouch. La sauvegarde se fait dans un type de fichier spécifique à AlgoTouch. Il est basé sur du xml, mais est modifié. Il est impossible de l'ouvrir avec un autre logiciel.


Charger un programme

Tout comme la sauvegarde, il est donc possible de charger un programme qui à été précédemment sauvegarder. Le type de fichier doit être le même que le celui de sauvegarde. Si le fichier est modifié, il se peut que le chargement ne fonctionne pas.


Lors du chargement d'un fichier, tous les blocs sont restaurés (même emplacement dans le programme. Le panneau de suivi des variables est lui aussi mis à jour lors du chargement.

Changer la langue

Pour le moment, seulement deux langues sont disponibles pour le logiciel AlgoTouch :

  • Français
  • Anglais

Lors du lancement de l'application, celle-ci récupère la langue du système s'adapte à celle-ci. Si la langue n'est pas supportée, le français sera mis par défaut.

Comment savoir s'il y a un erreur dans l'exécution du programme ?

Si le programme comporte une erreur, celle-ci sera récupérée lors de la compilation ou de l'exécution, puis affichée sous forme de popup.


Le message d'erreur est simplifié afin de ne garder que l'essentiel (et non toute la StackTrace de java).

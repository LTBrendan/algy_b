Si...Alors...Sinon

Si…Alors…Sinon est un bloc conditionnel :

  1. Si ‘condition respectée’
  2. Alors ‘effectuer actions’
  3. Sinon ‘effectuer autre action’

L'espace qui suit le Si permet de comparer facilement deux valeurs, de type entier ou charactère. Elles peuvent être des valeurs simples, ou des variables créées par l’utilisateur.


La première entrée accueille le premier élément à comparer.


Le second argument va lui accueillir le comparant, ou le comparé, en fonction ce qui a été rentré dans le 1.


Le premier argument d’état est celui situé après le Alors. Il ne sera exécuté que dans le cas où la condition donnée est vraie


Le second argument d’état est celui situé après le Sinon. Il ne sera exécuté que dans le cas où la condition donnée est fausse.

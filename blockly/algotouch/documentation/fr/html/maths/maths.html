Operations

Le bloc opérations permet d’effectuer les opérations de base sur les entiers. Les opérations disponibles sont : l’addition, la soustraction, la multiplication, la division et le reste de la division euclidienne.


La première entrée accueille le premier élément à calculer.


Le second argument va lui accueillir le second élément à calculer.


Pour les développeurs

Pour ce bloc, le type de sortie est :

  • operation_integer

Les types d'entrée sont :

  • integer
  • var_integer
  • char
  • var_char

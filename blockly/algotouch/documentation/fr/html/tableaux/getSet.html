Valeur de

Ce bloc permet de récupérer la valeur d'un tableau précédemment créé en fonction de son indice.


L'argument d'entrée est le tableau depuis lequel on veut récupérer la valeur.


Le second argument est l'indice auquel on veut récupérer la valeur. C'est soit un entier, soit un indice préalablement créé et associé à ce tableau.


Mettre a

Ce bloc permet de changer la valeur d’un tableau précédemment créé en fonction de son indice.


L'argument d'entrée est le tableau depuis lequel on veut récupérer la valeur.


Le second argument est l'indice auquel on veut récupérer la valeur. C'est soit un entier, soit un indice préalablement créé et associé à ce tableau.


Le troisième argument est la valeur que va prendre celles de l’indice souhaité dans le tableau. Elle peut être de type (en fonction du type du tableau) : char, var_char, integer, var_integer, operation_number ou indice.

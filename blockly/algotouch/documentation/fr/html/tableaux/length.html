Length

Ce bloc permet à l'utilisateur de connaitre la taille d'un tableau qu'il à précédemment créé. Il peut être utile notamment pour faire une boucle parcourant tout ou partie d'un tableau.


L'argument d'entrée correspond au bloc du tableau dont la taille est voulue. Il accepte donc les types char_array et integer_array, qui sont les deux types de tableaux possibles.

Le bloc renvoie donc une valeur, de type entier.

Pour les développeurs

Nom du bloc :

  • length_of

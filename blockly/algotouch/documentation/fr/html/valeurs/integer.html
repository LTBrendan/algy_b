Nombre entier

Permet à l’utilisateur de créer des valeurs de type entier (integer). Cependant il faut bien faire attention :


  • Ce n’est pas une variable pouvant être changée. Elle sert juste pour effectuer, par exemple, une comparaison avec une variable utilisateur.

  • La taille limite de cet entier est de 2147483647 et sa valeur minimum de -2147483648. (Attention au temps de calcul et aux erreurs de taille)

  • Seul un entier est accepté par le bloc. Si c’est un caractère quelconque qui est entré, il sera remplacé par la dernière valeur correct.

Ce block accepte seulement une seule entrée : un entier. Il est saisi directement au clavier par l’utilisateur.


Le bloc renvoie donc une valeur, de type entier.

Pour les développeurs

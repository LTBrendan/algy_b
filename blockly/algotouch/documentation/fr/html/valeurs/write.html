Ecrire valeur avec message

Permet à l’utilisateur d'afficher la valeur d'une variable pendant l’exécution du code. Cette valeur sera affichée, accompagnée éventuellement d'un message personnalisé, par le biais d’une « popup ».


Le premier argument va être le bloc de la variable qui va être affiché. Il ne peut s’agir que de blocs de type : var_char, var_integer, char_array ou encore integer_array.


Le premier argument d’état est tout simplement un champ de texte, afin de donner plus de précision à l’utilisateur sur la correspondance de la valeur qui va être affichée.

Pour les développeurs

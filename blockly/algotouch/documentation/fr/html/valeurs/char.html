Charactère

Permet à l’utilisateur de créer des valeurs de type char. Cependant il faut bien faire attention :


  • Ce n’est pas une variable pouvant être changée. Elle sert juste pour effectuer, par exemple, une comparaison avec une variable utilisateur

  • Sa taille est de 1. C’est un caractère, et non une chaine de caractères. Si plusieurs lettres sont entrées, seule la première sera conservée.

  • Le caractère est libre : il peut s’agir d’une lettre, d’un signe (=, &, _, …) ou encore d’un entier.

Ce block accepte seulement une seule entrée : un caractère. Il est saisi directement au clavier par l’utilisateur.


Le bloc renvoie donc une valeur, de type charactère.

Pour les développeurs

Lire valeur avec message

Permet à l’utilisateur de définir une valeur de variable seulement à l’exécution du code. Cette valeur sera attribuée à une variable créée, par le biais d’une « popup ».


Le premier argument va être le bloc de la variable qui va être modifiée par la valeur entrée par l’utilisateur. Il ne peut s’agir que de blocs de type : var_char ou var_integer.


Le premier argument d’état est tout simplement un champ de texte, afin de donner plus de précision à l’utilisateur sur la correspondance de la valeur qu’il va renseigner.

Pour les développeurs

Tableaux

Le logiciel AlgoTouch permet la création de tableaux, afin de pouvoir effectuer des traitements simples sur ceux-ci. Il est possible d'effectuer des recherches de valeurs, des suppressions de doublons...


Cependant le type du tableau doit être précisé à sa création. En effet, il peut contenir des caractères ou des entiers.



Une fois le tableau créé, plusieurs actions sont réalisables. En effet, il est possible de le supprimer ainsi que de le renommer. Une fois le tableau créé, il est possible de le modifier pour en changer la longueur ou bien insérer de nouvelles valeurs. Cela évite de créer des macros uniquement pour modifer un tableau déjà existant. Les autres actions que vous pourrez faire avec un tableau sont référencées dans les pages Taille et Valeur de, mettre à.


Pour les développeurs

Les tableaux créés sont de type :

  • char_array pour un tableau de charactères
  • integer_array pour un tableau d'entiers

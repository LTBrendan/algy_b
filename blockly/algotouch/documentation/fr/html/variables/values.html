Valeurs

Afin de pouvoir programmer des algorithmes intéressants, il est possible de créer des variables "locales". Celles-ci pourront être utilisables tout au long du programme, modifiées et utilisées avec des tableaux.


Le type de la variable est à choisir avant sa création. Elle peut être un charactère, un entier, un indice lié à un tableau (qui est de type entier) ou encore peut être définie en tant que constante, c'est à dire que c'est un entier qui ne pourra être modifié.



Dans le cadre d'une valeur de type entier, caractère ou index, il est possible par la suite d'en modifier la valeur pendant l'exécution du programme, ou encore de demander la valeur à l'utilisateur. Les valeurs crées peuvent être bien sûr supprimées ou renommées à volonté.



Une "popup" est lancée afin de renseigner le nom de la variable, et éventuellement sa valeur (dans le cadre d'une constante). Si la variable créée est un indice, le nom du tableau associé sera demandé s'il en existe plusieurs.


Pour plus de renseignement sur les accès aux tableaux, consultez la page Valeur de|Mettre a

Pour les développeurs

Les valeures créées sont de type :

  • var_char pour un charactère
  • var_integer pour un entier
